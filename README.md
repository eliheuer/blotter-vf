# Blotter
By Froyo Tam

![GIF IMAGE](https://github.com/eliheuer/blotter-vf/blob/master/docs/blotter_froyotam_preview.png)

## Build Instructions
```
python3 sources/BUILD.py --static --fontbakery --fixnonhinting
```
